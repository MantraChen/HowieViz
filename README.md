# HowieViz

Interactive data structure & algorithm visualizer for competitive programmers.

## Visualizers

- [x] Array
- [x] Stack
- [x] Queue
- [x] Linked List
- [x] Binary Search Tree
- [x] Binary Heap
- [x] Graph BFS/DFS
- [ ] Quicksort
- [ ] Merge Sort

## Getting Started

git clone https://github.com/MantraChen/HowieViz.git
cd HowieViz
npm install
npm run dev
