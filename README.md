# HowieViz

Interactive data structure & algorithm visualizer for competitive programmers.

## Visualizers

- [x] Array
- [ ] Stack
- [ ] Queue
- [ ] Linked List
- [ ] Binary Search Tree
- [ ] Binary Heap
- [ ] Graph BFS/DFS
- [ ] Quicksort
- [ ] Merge Sort

## Getting Started

git clone https://github.com/MantraChen/HowieViz.git
cd HowieViz
npm install
npm run dev
