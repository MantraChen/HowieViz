# HowieViz

Interactive data structure & algorithm visualizer for competitive programmers.

## Visualizers

### Linear
- [x] Array
- [x] Stack
- [x] Queue
- [x] Circular Queue
- [x] Deque
- [x] Linked List
- [x] Doubly Linked List

### Trees
- [x] Binary Heap
- [x] BST
- [x] AVL Tree
- [x] Trie
- [x] Segment Tree
- [x] Fenwick Tree
- [x] Red-Black Tree

### Graphs
- [x] Graph BFS / DFS
- [x] Dijkstra's
- [x] Bellman-Ford
- [x] Floyd-Warshall
- [x] Topological Sort
- [x] Prim's MST
- [x] Kruskal's MST
- [x] Union Find

### Sorting
- [x] Quicksort
- [x] Merge Sort
- [x] Heap Sort
- [x] Bubble Sort
- [x] Insertion Sort
- [x] Counting Sort
- [x] Radix Sort

### Searching
- [x] Binary Search
- [x] Linear Search

### Dynamic Programming
- [x] Fibonacci (Memoization)
- [x] 0/1 Knapsack
- [x] LCS
- [x] Edit Distance (Levenshtein)

## Getting Started

```bash
git clone https://github.com/MantraChen/HowieViz.git
cd HowieViz
npm install
npm run dev
```
